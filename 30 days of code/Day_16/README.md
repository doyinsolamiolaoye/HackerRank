# Exceptions - String to Integer

## Problem Statement
Read a string, `S` , and print its integer value; if `S` cannot be converted to an integer, print `Bad String`.

**Input Format**

A single string, `S`.

**Sample Input**
```
3
```

**Sample Output**
```
3
```

**Sample Input**
```
za
```

**Sample Output**
```
Bad String
```
