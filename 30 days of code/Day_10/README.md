# Binary Numbers

## Problem Statement
Given a base-`10` integer,`n` , convert it to binary (base-`2`). Then find and print the base- integer denoting the maximum number of consecutive `1`'s in `n`'s binary representation.

**Input Format**
A single integer, `n` .

**Sample Input**:
```
13
```

**Sample Output**:
```
2
```
